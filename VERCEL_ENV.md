# Переменные окружения для Vercel

После деплоя в Vercel, добавьте следующие переменные окружения в настройках проекта:

## Обязательные переменные:

1. `DEBUG` = `False` (для продакшена)
2. `SECRET_KEY` = `ваш-секретный-ключ` (сгенерируйте новый!)

## Как добавить переменные окружения:

1. Зайдите в настройки проекта на Vercel
2. Перейдите в раздел "Environment Variables"
3. Добавьте каждую переменную для Production окружения

## Генерация нового SECRET_KEY:

```python
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
```

## Важно:

- Никогда не коммитьте SECRET_KEY в репозиторий
- После добавления переменных, передеплойте проект
- Для локальной разработки используйте DEBUG=True
